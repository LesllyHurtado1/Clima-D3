#  Visualización de Datos Energéticos y Climáticos 

## Dataset

El dataset utilizado contiene cinco años de información (2020–2024) con registros diarios simulados que combinan indicadores climáticos globales y datos de consumo energético para aproximadamente 50 países.

Fue diseñado con fines de análisis y aprendizaje, especialmente para áreas como ciencia de datos, inteligencia artificial y estudios de sostenibilidad.

Los datos representan comportamientos realistas, incluyendo:

* Cambios de temperatura a lo largo del tiempo
* Variaciones en la actividad industrial
* Diferencias entre países
* Fluctuaciones en el consumo de energía
* Emisiones de CO₂ asociadas

Aunque los datos son sintéticos, están construidos de manera que imitan patrones reales, como ciclos estacionales y tendencias globales, lo que permite realizar análisis significativos sin necesidad de datos sensibles reales.

---

##  Visualizaciones incluidas

El proyecto contiene las siguientes gráficas:

* **Temperatura vs tiempo:** muestra cómo cambia la temperatura promedio global a lo largo de los años
* **Emisiones de CO₂ por país:** comparación entre países según sus emisiones promedio
* **Distribución del consumo energético:** análisis del comportamiento del consumo mediante boxplot
* **Energía renovable por país:** porcentaje de participación de energías limpias
* **Relación entre renovables y CO₂:** permite observar si existe alguna relación entre ambas variables


## 🎯 Objetivo 

El objetivo principal es observar patrones y comportamientos en los datos, como:

* Cambios a lo largo del tiempo
* Diferencias entre países
* Posibles relaciones entre variables
